Alpha?

I've only recently started working on the project.
Although the idea for it came about much earlier. There was even an idea to take the FreeBSD kernel.
I decided on the Linux, and the x86_64 architecture because it seems easier with the LinuxFromScratch project.

What's the plan for Alpha?

Try to finalize drivers and kernel configuration.
Basic console programs and maybe a simple package manager.
APFS support ‘from the box’, maybe only RO for first time.
WLAN improvement, ‘cause now there is a RTL & BRCM have problems.

Whats work now?

You can read the status of the system in the repository.
Saddenly, the public Alpha v0.2 only has files for QEMU or UTM VMs, but in a couple of versions
I think there will be a LiveCD ISO image ready, for boot up on real machine from USB stick or Ventoy.
